# Escolhendo o maior

![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/omeletao/assets/cover.jpg)

Leia 4 números e imprima o maior valor.

### Entrada

* Leia quatro valores inteiros do usuário.

### Saída

* Imprima o maior valor lido.

## Exemplos

<!-- load tests.toml --tests 2 -->
```py
>>>>>>>> INSERT
3
3
2
2
======== EXPECT
3
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
5
1
8
3
======== EXPECT
8
<<<<<<<< FINISH
```
